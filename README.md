# Pfetch

Lua dili ile yazılmış minimalist bir sistem bilgilendirme (fetch) aracı. Sadece GNU/Linux'ta çalışmaktadır. 
Hiçbir şekilde yapay zeka kullanılmamıştır.

## Kurulum
1. Bu depoyu klonlayın:
   git clone https://github.com/padros/pfetch
2. Dizine gidin:
   cd pfetch
3. Çalıştırın:
   lua pfetch
